# db0gj-tools
Repeater DB0GJ Tools - Sammlung verschiedener Tools

### Geplant ###
- Anbindung Standort DB0GJ via Hamnet
- Host vor Ort: Raspberry
- APRS iGate (RX only) mit SDR-RTL - spaeter RX-TX-Digi
- APRS Object Beacons
- APRS Telemetrie
